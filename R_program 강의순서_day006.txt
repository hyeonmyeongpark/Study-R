 8. R 데이터 시각화 : R의 기본으로 내장된 시각화 함수 plot()
   - 목적 : 비교, 구성, 분포, 관계
   1) plot()
      - 형식
        plot(y축 데이터, 옵션)
        plot(x축 데이터, y축 데이터, 옵션)
      - plot() : 산점도(scatter plot), 기본형
      - points() : 점을 그리는 함수
      - lines() : 꺽은선, 시계열 데이터
      - abline() : y = ax + b, 직선을 그리는 함수
      - curve() : 곡선을 그리는 함수
      - text() : 차트에 문자열을 표시하는 함수
      - polygon() : 다각형을 그리는 함수, 신뢰구간 같은 범위를 표현할 때 
      - arrows() : 화살표를 그릴 수 있는 함수
      - legend() : 범례
      - boxplot() : 상자수염그래프, 데이터의 분포를 보여주는 그래프
                   이상치를 확인
      - hist() : 히스토그램, 데이터의 분포를 보여주는 그래프

   2) 히스토그램(hist)
      - hist(데이터, 차트제목, x축제목, y축제목)
   3) 상자그림(boxplot)
   4) 산점도(point, scatter plot)
      - 산포도 : 대푯값을 중심으로 자료들이 흩어져 있는 정도를 시각화
      - 두 데이터 간에 상관관계의 유무를 시각적으로 표현하는 그래프
      - 형식
       plot(x축 자료, y축 자료, main=제목, sub=부제목, xlab=x축제목, ylab=y축제목,
            type=그래프종료, axes=테두리, col=색상, pch=점의모양, cex=점의크기)
       axes = F(테투리 없음), ann = F(축제목 없음)
   5) ggplot2 패키지
     - 패키지 설치 install.packages("ggplot2")
     - 패키지 로딩 library(ggplot2)
     - ggplot(data, aes()) + geom_point()
     - 표현하는 그래프 종류
       geom_point() : 산점도
       geom_histogram() : 히스토그램
       geom_boxplot() : 상자수염그래프
       geom_bar() : 막대그래프

       geom_line() : 선그래프

 10. R을 활용한 기초통계분석 : 추론통계분석
   1) 정규분포 : 세상에 거의 모든 분포
      - 자료의 분포가 평균을 중심으로 대칭을 이루는 분포(종모양)
      - 정규분포 난수
        rnorm(n, mean, sd)
      - 확률밀도함수
        dnorm(n, mean=0, sd=1, log=FALSE)
      - 누적분포난수 or 표준정규분포
        pnorm(n, mean=0, sd=1, lower.tail=T, log.p=F)
      - 표준점수 : z점수
        정규분포를 표준정규분포 단위로 맞추는 것
        z-value = x - mean(x) / sd(x)

  2) 가설검정
    - 가설검정 단계
      가설수립 => 표본으로부터 검정을 위한 통계량 계산 => 가설선택의 기준수립 => 판정
    
    - 모집단과 표본
      모집단 : 우리가 알고자 하는 대상 전체 즉, 조사 대상의 범위
      표본 : 모집단으로부터 조사하기 위해 선택된 조사 대상

    - 전수조사와 표본조사
      전수조사 : 모집단을 구성하고 있는 대상 전부를 조사하는 것
      표본조사 : 표본을 대상으로 조사

    - 통계분석 기법
      어떤 그룹, 집단, 형태등의 차이를 검정 : 차이검정
      => 단일표본 t검정(일표본 가설검정) : 표본이 1개인 경우

      => 대응표본 t검정 : 표본이 2개인 경우의 t검정
         독립표본 t검정 : 두 개의 독립적인 표본의 가설검정

      => 분산분석 ANOVA

      요소와 요소간의 인과관계(상관관계)를 파악
      => 상관분석 : 두 양적 자료 간에 통계적으로 유의한 관련성이 있는지를 검정하는 방법
         먼저 산점도(scatter plot)를 작성한 다음 두 양적 자료 간의 관계를 살펴본다.
         상관계수 : 두 양적 자료의 관련성 정도를 나타내는 수치값(-1 ~ 1)

      => 회귀분석 : 인과관계를 파악하는 방법으로 독립변수(설명변수)가 종속변수(반응변수)에
                   통계적으로 유의한 영향을 미치는지, 영향을 준다면 얼마나 주는지, 
                   최종적으로 회귀모형을 토대로 종속변수를 예측하기 위해서 사용하는 
                   통계분석 방법
          회귀방정식 y = ax + b
          단순회귀분석 : 독립변수 1개, 종속변수 1개
           결과값 <- lm(종속변수 ~ 독립변수, data=데이터셋이름)
           summary(결과값)

          다중회귀분석 : 독립변수 n개, 종속변수 1개
           lm(종속변수 ~ 독립변수1 + 독립변수2 + 독립변수3, data=데이터셋이름)
           lm(종속변수 ~ ., data=데이터셋이름)

    - 가설검정 : 모집단의 특성의 상태에 대한 주장인 가설에 대해 표본으로부터 얻은 정보를 바탕으로
               이를 채택할지 기각할지 판단함으로써 모집단의 상태에 대해 결정하는 과정
      => 귀무가설과 대립가설
         귀무가설(영가설) : 주로 차이가 없다. 두 집단은 같다
         대립가설(연구가설) : 연구자의 주장이 담긴 가설
         주로 귀무가설이 참이라는 가정하에 검정진행한다.(귀무가설이 기각됨을 증명)

    - p-value : 유의수준 or 유의확률
      도출한 분석모델의 결과가 통계적으로 의미가 있는지 없는지를 판단하는 지표
      가설의 채택과 기각을 판정하는 기준
      i) 사회과학분야 유의수준(임계값, 알파) - 0.05
          p < 0.05 95% 신뢰구간
      ii) 의생명분야 - 0.01   
          p < 0.01 99% 신뢰구간
      iii)
          p-value >= 0.05 귀무가설 채택(대립가설 기각)
          p-value < 0.05 귀무가설 기각(대립가설 채택)
