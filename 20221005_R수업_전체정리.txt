8. R을 활용한 기초통계분석 : 추론통계분석
1) 정규분포 : 세상에 고의 모든 분포
ㄴ 자료의 분포가 평균을 중심으로 대칭을 이루는 분포(종모양)
ㄴ 정규분포 난수
* rnorm(n, mean, sd) 
ㄴ 확률밀도함수
* dnorm(n, mean=0, sd=1, log=FALSE)
ㄴ 누적분포난수(표준정규분포)
* pnorm(n, mean=0, sd=1, lower.tail=T, log.p=F)
ㄴ 표준점수 :  z점수
정규분포를 표준정규분포 단위로 맞추는 것
z-value = x-mean(x) / sd(x)


1) 가설검정
ㄴ 가설검정 단계
* 가설수립 => 표본으로부터 검정을 위한 통계량 계산 => 가설선택의 기존수립 => 판정

ㄴ 모집단과 표본
* 모집단 : 우리가 알고자하는 대상 전체 즉, 도사 대상의 범위
* 표본 : 모집단으로부터 조사하기 위해 선택된 도사 대상

ㄴ 전수조사와 표본조사
* 전수조사 : 모집단을 구성하고 있는 대상 정부를 조사하는 것
* 표본조사 : 표본을 대상으로 조사

ㄴ 통계분석 기법
* 어떤 그룹, 집단, 형태등의 차이를 검정 : 차이검정
* 단일표본 t 검정  : 표본이 1개인 경우
* 대응표본 t 검정 : 표본이 2개인 경우의 t검정 ( 똑같은 집단 - 감기약 전/후)
* 분산분석 ANOVA

요소와 요소간의 인과관계(상관관계)를 파악
* 상관분석 : 두 양적 자료 간에 통계적으로 유의한 관련성이 있는지를 검정하는 방법
먼저 산점도(scatter plot)를 작성한 다음 두 양적 자료간의 관계를 살펴본다.
상관계수 : 두 양적 자료의 관련성 정도를 나타내는 수치값(-1 ~ 1)

* 회귀분석 : 인과관계를 파악하는 방법으로 독립변수(설명변수)가 종속변수(반응변수)에 통계적으로 유의한 영향을 미치는지, 영향을 준다면
얼마나 주는지, 최종적으로 회귀모형을 토대로 종속변수를 예측하기 위해서 사용하는 통계분석 방법

회귀방적식 y = ax + b
단순 회귀분석 : 독립변수 1개, 종속변수 1개
lm(종속변수 ~ 독립변수, data=데이터셋이름)

다중 회귀분석 : 독립변수 n개, 종속변수 1개
lm(종속변수 ~ 독립변수1+독립변수2 + 독립변수3, data=데이터셋이름)
lm(종속변수 ~ ., data=데이터셋이름)


- 가설검정 : 모집단의 특성의 상태에 대한 주장인 가서에 대해 표본으로부터 얻은 정보를 바탕으로
이를 채택할지 기각할지 판단함으로써 모집단의 상태에 대해 결절하는 과정
* 귀무가설과 대립가설
귀무가설(영가설) : 주로 차이가 없다. 두 집단은 같다
대립가설(연구가설) : 연구자의 주장이 담긴 가설
주로 귀무가설이 참이라는 가정하에 검정진행한다.(귀무가설이 기각됨을 증명)

- p-value : 유의수준 or 유의확률
 도출한 분석모델의 결과가 통계적으로 의미가 있는지 없는지를 판단하는 지표
(가설의 채택여부를 판정하는 기준)

1) 사회과학분야 유의수준(임계값, 알파) - 0.05
p < 0.05 95% 신뢰구간
2) 의생명분야 - 0.01
p < 0.01 99% 신뢰구간
3) p-value >= 0.05 귀무가설 채택(대립가설 기각)
p-value< 0.05 귀무가설 기각(대립가설 채택)
